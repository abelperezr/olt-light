# Guía de CLI

:::info En construcción

Esta sección está en desarrollo. Documentará el uso del eCLI incluido en
Light OLT, con ejemplos de navegación, consulta, configuración, commit y cambio
entre los planos SHELF, LT e IHUB.

:::

La futura guía describirá exclusivamente el CLI implementado por este proyecto.
No es, ni sustituye, la guía oficial de configuración del eCLI de Nokia
Lightspan. La documentación oficial del fabricante debe obtenerse a través de
Nokia y no se incluye en este repositorio.

Mientras se completa esta página, consulta
[Personalizar el eCLI](../how-to/customize-ecli.md) para conocer el código y el
ciclo de reconstrucción.
